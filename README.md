## 🏁 Starting

```
# Clone this project
$ git clone https://github.com/mmazitov/miro

# Access
$ cd miro

# Install the dependencies using Yarn or npm:
$ $ yarn install or $ npm install

# Run the project
$ $ yarn start or $ npm start

# The server will initialize in the <http://localhost:3000>
```
